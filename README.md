# stupid_meter_RTOS
A low-cost USB voltmeter with screen

MCU: AT32F421G8U7

Current sampling: INA226

Screen: ST7735

DC-DC buck: MP2451

2-Layer board

Refer to bare-metal version: https://github.com/wwng2333/stupid_meter/tree/RevB-AT32F421
